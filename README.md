# Teslo API

1. Clonar proyecto
2. ``` yarn install ```
3. Clonar el archivo ``` .env.template ``` y renombrarlo a ``` .env ```
4. Cambiar las varibles de entorno
5. Levantar la base de datos

1. Levantar la base de datos
```
docker-compose up
```

6. Levantar: ``` yarn start:dev ```
